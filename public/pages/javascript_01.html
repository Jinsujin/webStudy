자바스크립트 포함하는 방법

head 안에 script src="../lib/main.js"

HTML parsing -> [js fetch -> js 실행] -> 다시 HTML parsing

body 끝나기 전에 추가

HTML parsing ->[페이지 준비됨]-> fetch js -> js 실행

javascript 파일에 의존적이라면,
페이지가 로드됬어도, 자바스크립트 파일이 로드 될때까지 기다려야 한다.

head 안에 async 키워드로 포함

HTML parsing / fetch js -> [ js실행 / HTML parsing STOP ] -> parsing HTML

자바스크립트 파일 다운로드 받는 시간 절약
먼저 가져온 js 를 먼저 실행. 실행순서 보장되지 않음

defer 키워드로 포함

HTML parsing/ fetch js -> 페이지가 준비되었을때 js 실행

파싱하는 동안 js 파일을 병렬로 가져오고, 페이지가 준비 되었을때 모든 js 파일을 순차적으로 실행
